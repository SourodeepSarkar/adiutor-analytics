Workflow

Export analytics from Adiutor.

Access locally stored study snapshots and import them into the analytics dashboard for visualization and reports.

Step 01

Open Settings

Launch Adiutor, then click Adiutor → Settings from the macOS menu bar. You can also use the standard macOS shortcut: ⌘,

Step 02

Locate Statistics Export

Scroll to the bottom of the Settings window where the statistics export controls are available.

Step 03

Open Statistics Folder

Click the Open Statistics Folder button. Adiutor will directly open the Finder directory containing exported JSON analytics snapshots.

Step 04

Import JSON Snapshots

Drag exported JSON files into the upload area above or use the file picker to instantly generate analytics charts, usage insights, and reports.
